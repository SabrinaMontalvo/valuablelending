I’m committed to providing an accessible experience for all users, including those using assistive technologies.

Standards

We aim to follow WCAG 2.1 AA where feasible: meaningful HTML, sufficient color contrast, keyboard access, focus visibility, and text alternatives.

Feedback

If you encounter a barrier, please email hello@sabrinamontalvo.com with a description of the issue and page URL. I’ll do my best to fix it promptly.

Compatibility

This site is built to work on modern browsers and mobile devices. Some features are available offline using the installed PWA.
